이벤트 처리하기

기본 이벤트 속성

  • focus : input tag에 포커싱 되었을때
  • blur : input 태그에서 다른 input 태그로 포커싱을 옮길때 포커싱이 사라질때.
  • 중요!! : 동일한 태그에 focus와 blur를 같이 걸면 무한반복...
  • change : input 태그에 있는 내용이 변경 됐을때 select등 선택했을 때
  • click : 마우스를 클릭했을 때
  • dblclick : 마우스 더블 클릭 시
  • mouseover : 마우스가 올라왔을 때 (hover와 비슷)
  • mouseenter : 마우스가 tag안에 들어왔을 때
  • mouseleave : 마우스가 tag에서 나갔을 때
  • keydown : 키보드 눌렀을 때
  • keyup : 키보드 올라올 때
  • keypress : 키보드를 눌렀다가 올라왔을 때
  • submit : 데이터 전송
  • reset : 데이터(form)이 초기화
  • load : 문서가 로딩된 후
  • resize : 창의 크기가 변동 됐을 때
  • scroll : 스크롤 바를 조작했을 때

이벤트 적용하기

페이지에서 이벤트를 적용해 보자. 모든 태그에서 이벤트를 사용할 수 있음! 표준방식으로 설정하는 것을 권고함! addEventListener("이벤트명", 실행함수);
