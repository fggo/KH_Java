Regular Expression

regex 객체 생성

대체문자 변환해주는 문자

$& : 제일 처음 일치하는 문자열 $` : 매칭된 문자열의 앞 문자열 #\$ : 매칭된 문자열의 뒤문자열

플래그 문자

문자열 검색에 대한 옵션을 지정하는 것
g : 전역비교를 수행, * 설정하기 전 한개 매칭되면 검색종료!
i : 대소문자 구문하지 않음
m : 여러줄 검색, 개행을 기준으로 다시 시작함

메타문자

앵커문자

^ : 문자열 앞에 오는 문자를 찾는 것
$ : 문자열 뒤에 오는 문자를 찾는 것

이스케이프 문

예약어를 문자로 쓰고 싶을때 사용가는것 (. ^ \) 역슬래시

포인트 .

임의의 한 글자. 자리수 한자리 차지하는 것.

[]대괄호

한문자, 묶음안의 내용은 범위를 나타냄 대괄호 안의 ^ 시작이 아니라 예외구문

()와 |

여러문자열 중에 해당하는 것을 검색하려면 (문자열|문자열2|문자열3)식으로 사용가능.

추가 메타 문자

\d : 모든숫자(==[0-9])
\D : 숫자를 제외한 문자
\w : 영어,숫자,_(언더바) 한글/공백 제외
\W : 영어,숫자,_(언더바)를 제외한 문자
\s : 공백문자(띄어쓰기, 탭, 줄바꿈)
\S : 공백문자 아닌 문자

수량 문자

a* : a가 0개 이상
a+ : a가 1개 이상
a? : a가 0또는 1개

{}를 이용한 수량표시

a{5} : a가 5개 이상
a{3,5} : a가 3~5개
a{2,} : a가 2개 이상
