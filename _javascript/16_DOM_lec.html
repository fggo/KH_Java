DOM(Document Object Model)

html소스코드의 태그를 객체화하여 관리하는 것 Document객체를 이용하여 각 태그 element에 접근할 수 있고 새로운 element를 생성할 수 있음

새로운 element생성

페이지에 동적으로 새로운 태그를 의미함. 사용자의 event에 의해 생성하는 것 생성절차!!!!! 1. document.createElement()함수를 이용하여 태그 element를 생성(node) 2. 태그 element에 연결할 textNode도 생성 document.createTextNode()함수 이용 3. 새로 생성된 태그 element하고 textnode를 연결 appendChild()함수를 통해 연결 -연결될 객체명(새로생성된 element).appendChild(연결할 객체(textNode)); 4. 새로생성된 태그 element를 html문서 구조에 연결 body태그나 body태그의 자식태그들한테 연결함. appendChild()함수이용
